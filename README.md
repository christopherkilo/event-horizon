# Event Horizon

Premium event discovery platform frontend — Next.js, TypeScript, Tailwind, Framer Motion.

## Pages

- `/` Home
- `/browse` Search + filters
- `/events/[id]` Event details
- `/favorites` Saved events
- `/about` About

## Run

```bash
npm install
npm run dev -- -p 3002
```

## Data

Edit placeholder events in `lib/eventData.ts`.
